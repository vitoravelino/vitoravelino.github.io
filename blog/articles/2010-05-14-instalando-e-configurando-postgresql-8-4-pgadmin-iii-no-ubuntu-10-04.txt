Mais um guia aqui no blog. Este foi devido a necessidade da instalação do PostgreSQL e pgAdmin para o projeto da disciplina de SIG (Sistema de Informação Geográfica) que trabalhará juntamente com o PostGIS para o armazenamento e consulta de informações espaciais. Em um próximo post irei explicar como instalar a extensão do PostGIS para o PostgreSQL.

Vamos lá!

### Primeiro passo:

<pre>
    <code>
$ sudo apt-get install postgresql-8.4 pgadmin3
    </code>
</pre>

Instalação do PostgreSQL 8.4 + pgAdmin III.

### Segundo passo:
<pre>
    <code>
$ sudo -u postgres psql template1
- template1=# ALTER USER postgres with encrypted password 'sua_senha';
- template1=# \q
    </code>
</pre>

Defina uma senha para a conexão com o banco de dados. Ela será importante para acessar o banco via pgAdmin como veremos mais adiante.

### Terceiro passo (opcional):

Caso você tenha a necessidade de acessar o banco de uma outra máquina a não ser o localhost, abra o arquivo <em>/etc/postgresql/8.4/main/postgresql.conf</em> e modifique a linha

<pre>
    <code>
#listen_addresses = 'localhost'
    </code>
</pre>

por

<pre>
    <code>
listen_addresses = '*'
    </code>
</pre>

ou

<pre>
    <code>
listen_addresses = 'ip'
    </code>
</pre>

Neste momento já temos o PostgreSQL e pgAdmin instalados corretamente e funcionando. Até agora nada de utilização do pgAdmin, veremos como conectar-se ao PostgreSQL instalado nos passos seguintes.

### Quarto passo:

Abra o pgAdmin e você terá uma tela como a seguinte:

Adicione uma nova conexão e preencha os dados *Name*,* Host*, e* Password *(o mesmo setado no passo dois deste guia) corretamente como a imagem abaixo:

<a href="http://cloudfront.vitoravelino.net/images/articles/pgAdmin2.png" class="fancybox" title="pgAdmin - Nova conexão"><img src="http://cloudfront.vitoravelino.net/images/articles/pgAdmin2.png" alt="pgAdmin - Nova conexão" width="300" height="343" /></a>

<a href="http://cloudfront.vitoravelino.net/images/articles/pgAdmin3.png" class="fancybox" title="pgAdmin - Pós-criação de conexão"><img src="http://cloudfront.vitoravelino.net/images/articles/pgAdmin3.png" alt="pgAdmin - Pós-criação de conexão" width="300" height="231" /></a>

Pronto! Agora você está apto a por a mão na massa e criar suas tabelas.
